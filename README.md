# Monorepo

Reusable Bun monorepo conventions, package skeletons, repository tooling, shared
configuration, and domain-neutral TypeScript packages.

## Requirements

- Bun 1.3.14

## Setup

```sh
bun install
bun run verify
```

## Repository commands

```sh
bun run format
bun run lint
bun run compile
bun run test
bun run verify
```

Package structure and implementation rules are documented in
[`docs/conventions`](./docs/conventions/README.md).
