# Repo CLI

CLI package for repository operations.

## Scripts

- `bun run cli --help`
- `bun run cli compile`
- `bun run cli format`
- `bun run cli health`
- `bun run cli lint`
- `bun run cli skeleton --packages clis --name behavior-datasets`
- `bun run cli test`
- `bun run cli verify`
- `bun run cli verify deps`
- `bun run cli verify package-exports`
- `bun run verify`
