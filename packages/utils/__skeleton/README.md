# Util \_\_skeleton

Copy this folder when creating a new `packages/utils/*` package.

After copying:

1. Rename the folder.
2. Replace `@jearle/util-__skeleton` in `package.json`.
3. Replace `__skeleton` and related exported identifiers.
4. Rename `src/__skeleton`.
5. Rename `src/__skeleton/create-__skeleton.ts`.
6. Replace exported identifiers such as `create__skeleton`, `PropsCreate__skeleton`, `ResultCreate__skeleton`, and `__skeletonResult`.
7. Update imports and exports in `src/index.ts` and `src/__skeleton/index.ts`.
8. Replace or remove the smoke-test implementation with utility behavior.
9. Run `rg "__skeleton"` from the copied package and replace any remaining scaffold markers.
