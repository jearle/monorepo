# Branding

- Spell the company and product name as `Monorepo` in human-facing prose.
- Use lowercase `monorepo` only for code identifiers, package names, commands, file paths, environment variables, and other machine-readable names where lowercase is required or already established.

# Repository Conventions

- Read `docs/conventions/README.md` before changing TypeScript structure, naming, dependency wiring, services, or command/app composition.
- Repository conventions override `~/.ai/ts-coding-standards.md` and broader docs when they are more specific.
